\subsection{WORK REMAINING}
\begin{itemize}
    \item \textbf{Implementation of AI}: 
    The current system operates on threshold-based control. The next stage involves training and deploying machine learning models for predictive analytics. This will enable early detection of anomalies (e.g., abnormal gas levels, irregular growth), optimization of feeding schedules, and adaptive control of temperature and humidity in real time. Integration of these AI modules into the cloud dashboard is also required.

    \item \textbf{Validation and testing}: 
    While preliminary tests in a simulated environment are complete, extensive field validation in an actual poultry farm remains. This phase will include comparing sensor data with standard instruments, stress-testing system reliability over long durations, and verifying AI predictions against real-world conditions. The outcome will confirm the accuracy, robustness, and practical applicability of SmartFlock.
\end{itemize}
