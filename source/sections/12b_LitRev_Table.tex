\subsection{Synthesis for Current Project}

The literature supports the following approach for the GJ Poultry project:

\begin{enumerate}
    \item \textbf{Environmental Monitoring:} Multi-sensor framework with temperature, humidity, CO$_2$, and NH$_3$ sensors \cite{lashari2023internet, chang2023iot, godinho2025wireless}

    \item \textbf{Data Acquisition:} ESP8266/ESP32-based IoT platform with cloud connectivity \cite{kamor2024poultry, yourstory2018poultrymon}

    \item \textbf{Anomaly Detection:} Isolation Forest for environmental anomalies \cite{liu2008iforest} combined with Random Forest classification \cite{breiman2001rf}

    \item \textbf{Sensor Validation:} Bland-Altman analysis and ICC for reliability testing \cite{bland1986agreement, koo2016icc}

    \item \textbf{Predictive Analytics:} ML-based prediction following \cite{ji2025egg}'s approach for production metrics

    \item \textbf{Farm Management:} Embedded + mobile platform for accessibility \cite{karun2024iot}
\end{enumerate}

\subsection{Research Gaps Addressed}

\begin{itemize}
    \item Integration of multiple monitoring subsystems (behavior + environment + weight) as identified by \cite{ojo2022internet}
    \item Cost-effective solutions for small farmers in developing countries \cite{karun2024iot}
    \item Localized research for environmental/economic contexts (e.g., Nepal)
    \item Validation methodology using established statistical frameworks \cite{bland1986agreement, koo2016icc}
\end{itemize}
